# DeltaDotta trademark guidelines

The names **DeltaDotta** and the DeltaDotta logo identify this project and its
official releases. They are project trademarks, whether or not registered in a
particular jurisdiction.

You may use the name and unmodified logo to truthfully refer to DeltaDotta or to
describe compatible integrations. Do not use them in a way that implies
endorsement, sponsorship, or an official relationship. Do not use the name or
logo as the primary brand for a fork, service, domain, or product without written
permission from the project maintainers.

Forks should use a distinct name and visual identity while retaining the
attribution required by the [Apache-2.0 license](LICENSE). These guidelines do
not limit rights granted by that license for the source code itself.
